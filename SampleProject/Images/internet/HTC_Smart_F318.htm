HTC Smart F318

  • 1. Slide down to unlock touch screen, then press arrow back button to pop up Main Menu.
  • 2. From the main menu>>> go to>>>> Settings>>>then Wireless Controls.
  • 3. Inside Wireless controls select Mobile Network Settings
  • 4. Select Access point names then press small silver button below touch screen to pop up Options and select New and enter the following details:

    • Name: Bmobile Internet
    • APN: internet
    • Username: leave empty
    • Password: leave empty
    • Authentication: PAP
    • APN DNS 1: leave empty
    • APN DNS 2: leave empty
    • Proxy Server: leave empty
    • Proxy port: leave empty
    • Proxy password: leave empty
    • Proxy type: leave empty
