HTC A9191 Desire HD

Start screen > Choose the arrow symbol > Choose Settings > Choose Wireless & networks > Choose Mobile network, and check this option. Choose Mobile networks > Choose Access Point Names > Choose the menu key, and choose New APN.

Under Name: enter Bmobile Internet.
Under APN: enter internet
Under Proxy: enter nothing.
Under Port: enter nothing.
Under User name: enter nothing.
Under Password: enter nothing.
Scroll down.
Under MCC: enter 537.
Under MNC: enter 01.
Under Authentication type, choose PAP.
Under APN type: enter default. (Note: in lower case)
Choose the menu key.
Choose Save.
Choose Bmobile Internet, and check this option.
Return to the Main menu.

Choose Internet > Choose the menu key > choose More > and then Settings > Choose Set homepage and Choose Set home page Enter http://www.bmobile.com.pg
Return to the Start screen.
Start screen.
Switch off the HTC Desire HD and restart it in order to activate the settings.
You can now start using the internet.
