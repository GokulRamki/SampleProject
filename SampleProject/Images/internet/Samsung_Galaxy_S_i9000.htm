Samsung Galaxy S i9000

  • Touch the Menu key
  • Touch Settings
  • Touch Wireless & networks
  • Touch Mobile networks
  • Touch Access Point Names
  • Touch the Menu key
  • Touch New APN
  • Touch Name Enter Bmobile Internet, Touch OK
  • Touch APN Enter: “internet”
  • Touch OK
  • Touch Proxy Enter nothing, Touch OK
  • Touch Port Enter nothing, Touch OK
  • Touch Username Enter nothing, Touch OK
  • Touch Password Enter nothing, Touch OK
  • Touch the Menu key Touch Save
  • Touch the circle to the right of the WEB profile name you have just created to turn it green Press the Home key to return to the home screen
  • Touch Applications
  • Touch Browser
  • Touch the Menu key
  • Touch More
  • Touch Settings
  • Scroll to and touch Set home page Enter http://www.bmobile.com
